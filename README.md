# Image Viewer
Receives images via postMessage <br>
Made using React + Vite + TS <br>
Utilizes a Manifest v3 Chrome extension <br>

# Build Instructions
## build
npm install <br>
npm run build <br>
## dev
npm run dev <br>
or <br>
npm run dev -- --host <br>

# Deploy Instructions
npm run deploy

# Chrome Extension Install
- Browser settings 
- Extensions 
- Manage Extensions 
- Enable Developer Mode 
- Load Unpacked
- Use /Extension Folder
